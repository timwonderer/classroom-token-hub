# Onboarding and Feature Settings Troubleshooting

Diagnostic guide for resolving issues with missing features and the getting started widget.

## Feature is missing for students

### Symptoms
- Students cannot see a specific feature (like Store or Rent) in their navigation menu.
- The feature pages do not load for students or show an access error.

### Causes & Solutions
**Cause 1: Feature disabled in settings**
- **Check:** View the Feature Settings page for the specific class period.
- **Fix:** Toggle the feature to "ON". Features can be disabled per class period; period-specific settings override global defaults.

**Cause 2: Student in wrong class context**
- **Check:** Ask the student which class is currently selected in their dashboard switcher.
- **Fix:** Students must be in the correct class context to see features that are enabled for that specific period but disabled for another.

## Getting started widget is gone

### Symptoms
- The setup checklist widget is no longer visible on the dashboard.

### Causes & Solutions
**Cause 1: Widget was dismissed**
- **Check:** The widget disappears automatically after you dismiss it or complete all tasks.
- **Fix:** You can show it again from **Account Personalization** (Settings -> "Show Getting Started Widget" button).

## When to Contact Support
Report this issue if:
- A disabled feature still appears and is functional for students (this indicates a settings sync bug).
- The settings page fails to save with a database error.
