# Feature Toggles

## Overview
Feature toggles let you customize a classroom economy by class context. In v2.0, feature behavior should be understood as class-scoped where applicable, not teacher-global across every join code.

## Step-by-Step Instructions

### Toggling Features
1. Navigate to **Settings > Features** in the teacher sidebar.
2. Confirm you are in the correct class using the current class selector.
3. Review the list of available modules (e.g., Banking, Rent, Store).
4. Toggle features on or off depending on what you plan to use with that class.
5. Changes are applied immediately and will affect that class context for both you and your students.

## Important Notes
> [!TIP]
> **Start Simple:** If you are new to classroom economies, start with just Payroll and Store unlocked. Turn on Rent and Banking once your students are comfortable with the basics.

> [!IMPORTANT]
> **Class context matters:** If you teach multiple classes, verify your current class before changing feature toggles so you do not mistake one class's setup for another.
