# Troubleshooting Hall Passes

If something doesn't look right with your Hall Passes, check this guide first before reporting an issue to your teacher.

## If your request is denied or disappears, check these first:
- Hall pass is enabled for your class period.
- You have hall passes available (some teachers require passes for approval).
- The request was rejected or cancelled (pending passes can be cancelled).

## This is normal and expected when:
- A teacher approves a pass and a pass number appears.
- Office, Summons, or "Done for the day" passes do not deduct from your balance.
- After approval, you check yourself out from your dashboard instead of using a hallway terminal.
- Your pass changes from approved to returned only after you check out and your teacher marks you back in.

## When to report an issue to your teacher
If you've checked everything above and the problem is still occurring:
- Tell them the pass status (pending, approved, left, returned).
- Tell them the reason selected for the pass.
- Use the "Report Issue" button on the Help page.
