# Track Redemptions

Use this guide to check whether a store redemption is pending or completed.

## Where to work
Go to **Store** and review your recent purchases.

## What to look for
- Pending items that require teacher approval.
- Completed items that are ready to claim.

## If a redemption is stuck
Submit an issue report from Help and Support.
