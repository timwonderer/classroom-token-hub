# Attendance History and Pay Status

Use Work and Pay to review your attendance record and projected earnings.

## Where to work
Go to **Work and Pay** from the student sidebar.

## What you can see
- Attendance records for each class block.
- Unpaid time totals.
- Projected pay based on current settings.

## If something looks wrong
Use the help icon on an attendance record to report a problem.
