# Accounts and Transfers

Use Finances to view balances, transfer funds, and review transactions.

## Where to work
Go to **Accounts** from the student sidebar.

## View balances
The Accounts tab shows checking, savings, and total balance.

## Transfer funds
Use the Transfer tab to move money between checking and savings.

## Review transactions
Use the Transactions tab to review recent activity.
