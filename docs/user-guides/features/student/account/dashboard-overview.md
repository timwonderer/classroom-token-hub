# Student Dashboard Overview

Your dashboard summarizes balances, work status, and quick links.

## Where to work
Go to **Dashboard** from the student sidebar.

## What you will see
- Current balances and recent activity.
- Work and pay status.
- Quick links to Store and Bills.
