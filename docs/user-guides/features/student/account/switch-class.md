# Switch Classes

If you are in more than one class, use the class switcher in the sidebar.

## Where to work
Use the **Current Class** drop-down in the student sidebar.

## How it works
Select the class you want. The dashboard updates to the selected class.

## If a class is missing
Use **Add Class** with the join code from your teacher.
