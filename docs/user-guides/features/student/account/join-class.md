# Join or Add a Class

Use this page when your teacher gives you a join code for an additional class.

## Where to work
Go to **Add Class** from the student sidebar.

## What you need
- Join code from your teacher.
- Your first initial, last name, and date of birth.

## Add the class
1. Enter the join code.
2. Enter your name and date of birth exactly as your teacher entered them.
3. Submit the form to add the class.
