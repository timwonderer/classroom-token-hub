# Join or Add a Class

## Overview
If you are already part of one classroom economy but need to join another (for example, if a different teacher also uses the platform), you can add a new class using a direct join code.

## Step-by-Step Instructions

### Adding a New Class
1. Go to **Add Class** from the student sidebar.
2. Enter the unique **Join Code** provided by your new teacher.
3. Enter your first name, last name, and date of birth. **Important:** These must be typed exactly as your new teacher entered them into their roster.
4. Submit the form to link the new class to your existing account.

## Important Notes
> [!NOTE]
> **Matching details:** If the system says you cannot be found, it usually means you misspelled your name or your teacher made a typo when creating your account. Ask your teacher to double-check their Class Roster.
