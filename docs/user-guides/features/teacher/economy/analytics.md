# Analytics Dashboard

Analytics summarize system health and economy behavior across your classes.

## Where to work
Go to **Economy > Analytics**.

## What you will see
- System health signals and alerts.
- Economy trends for balances, rent, and payroll.
- Student and class level summaries.

Use the detailed analytics guide for definitions and interpretation tips.
