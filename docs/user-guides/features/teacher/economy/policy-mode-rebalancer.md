# Economy Policy and Rebalancing

## Overview

Economy policy mode helps you choose how much budgeting pressure or flexibility you want in a class economy. The rebalance review then shows suggested updates for supported settings so your current setup better matches that policy.

Use this workflow when you want to:

- make the economy tighter or more forgiving
- compare current rent or insurance values to the active policy
- apply suggested updates now or wait until the next payroll run

## Before you start

1. Configure payroll settings for the class you want to review.
2. Open the correct class period before changing economy settings.
3. Remember that changing policy mode does not rewrite past transactions.

## Change the policy mode

1. Open **Economy > Economy Health**.
2. Select the class period you want to review.
3. Choose the policy mode that best fits your classroom:
   - **Tight** for more budgeting pressure
   - **Default** for a balanced baseline
   - **Comfortable** for more breathing room
4. Save or confirm the policy change.

After the change, Economy Health uses the new policy bands for recommendations and alignment checks.

## Review rebalance suggestions

1. Stay on **Economy Health** after updating the policy.
2. Open the rebalance review when prompted.
3. Compare the current value and suggested value for each supported setting.
4. Select only the changes you want to apply.

If a setting does not appear in the rebalance review, update it manually in its own settings page.

## Activation choices

When you apply selected changes, you may be able to choose one of these activation options:

- **Apply now**: updates the selected settings immediately
- **Next payroll**: stores the selected changes and applies them during the next payroll run

Use **Apply now** when you want the economy to shift immediately.

Use **Next payroll** when you want the change to line up with a pay cycle.

## Important notes

> [!NOTE]
> Policy mode changes update the recommendation profile, not your historical ledger.

> [!TIP]
> If you are unsure which mode to choose, start with **Default**, review affordability and savings pressure in Economy Health, and then move tighter or more comfortable if needed.

> [!IMPORTANT]
> Scheduled rebalances depend on a later payroll run. If payroll is not run, the pending changes do not activate yet.
