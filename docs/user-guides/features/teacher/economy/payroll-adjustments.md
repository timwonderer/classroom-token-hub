# Payroll Adjustments

## Overview
Payroll adjustsments allow you to configure standardized rewards and fines that can be applied to student accounts. While regular payroll handles attendance-based earnings, adjustments let you incentivize behavior or enforce classroom rules consistently.

## Step-by-Step Instructions

### Managing Rewards and Fines
1. Navigate to **Economy > Payroll** in the teacher sidebar.
2. Click on the **Rewards** or **Fines** tab depending on what you want to configure.
3. Use the **Add Reward** or **Add Fine** button to create a new adjustment. You'll need to provide:
   - **Name**: A short, clear description (e.g., "Helping a classmate", "Missing assignment")
   - **Amount**: The monetary value of the adjustment
   - **Description**: (Optional) Extra details about when to apply this
   - **Active Status**: Whether this adjustment is currently available for use
4. To remove an adjustment, use the delete or archive option next to the item in the list.

### Applying Adjustments
Once configured, these standardized rewards and fines can be quickly applied to individual students or groups when processing transactions or managing accounts, keeping your classroom economy consistent and fair.

## Important Notes
> [!IMPORTANT]
> **Balance is Key:** Ensure your fines are not overly punitive relative to the Classroom Wage Index (CWI), and your rewards don't cause runaway inflation. Use the [Economy Health](economy-health.md) page to monitor standard fine amounts.
