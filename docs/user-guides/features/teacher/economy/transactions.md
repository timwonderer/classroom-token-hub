# Transactions

## Overview
The Banking page includes transaction history so you can review deposits, withdrawals, fees, and interest payouts in one place.

## Step-by-Step Instructions

### Reviewing and Voiding Transactions
1. Navigate to **Economy > Banking** in the teacher sidebar.
2. Open the transaction history section or tab within Banking.
3. Use the available filters for student, block, type, and date range to locate a specific entry.
4. If you find an incorrect entry, select **Void** on the transaction row to remove its financial impact.

## Important Notes
> [!NOTE]
> Voided transactions remain visible in the ledger for transparency and audit history, but their values are no longer counted in the student's balance.
