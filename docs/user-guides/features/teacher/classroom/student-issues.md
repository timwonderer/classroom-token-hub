# Student Issues Queue

Students can report issues from their Help & Support menu. Review them in the queue.

## Where to work
Go to **Classroom > Student Issues**.

## Triage workflow
1. Open the issue to see the details.
2. Verify the related transaction or attendance record.
3. Resolve the issue and reply if needed.

## What students can report
Common reports include attendance errors, transaction problems, store issues, and login trouble.
