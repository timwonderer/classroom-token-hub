# Class Setup and Join Codes

This page covers class sections, join codes, and the roster data students must match when they claim accounts.

## Where to work
Go to **Classroom > Students** and open the **Class Roster** tab.

## Create or confirm class sections
1. Open **Class Roster**.
2. Use the class selector to view existing sections.
3. Create the section you need before adding or importing students.

## Join codes
Each class section has a join code. Students must enter the join code when claiming or adding the class.

## Keep roster data aligned
The roster fields you enter are the exact values students must use to claim accounts. The key fields are:

- First name
- Last name
- Date of birth
- Class section

## When students cannot join
If a student cannot join, confirm the join code and confirm their roster details match exactly.
