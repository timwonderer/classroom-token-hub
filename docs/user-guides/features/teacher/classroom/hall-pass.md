# Hall Pass

Use the Hall Pass queue to approve, deny, or close student hall pass requests.

## Where to work
Go to **Classroom > Hall Pass**.

## Review the queue
The queue shows active requests. Open a request to see details before approving or denying.

## Close a pass
Close a pass when the student returns. Closing keeps the request history accurate.

## Troubleshooting
If requests are not showing, check the diagnostics guide.
