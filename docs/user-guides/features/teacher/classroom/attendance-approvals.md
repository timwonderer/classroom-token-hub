# Attendance and Approvals

## Overview
The **Attendance** page lets you review the time-tracking records used by payroll. You can also enable or pause attendance taps by class block.

## Step-by-Step Instructions

### Controlling Attendance Taps
1. Open **Classroom > Attendance** in the teacher navigation.
2. Use the class/block toggles to enable or pause **Start Work** and **Break / Done** taps.
3. Pause taps when class is not meeting or when you want to enforce a hard cutoff.

### Reviewing Attendance History
1. In the **Attendance History** panel, use filters for period, class, status, and date range.
2. Click **Apply** to refresh records.
3. Review the latest events and totals for start and end taps.

## Important Notes
> [!TIP]
> **Missing data:** If records appear missing, clear filters and widen the date range first.

> [!IMPORTANT]
> **Attendance history is append-only:** Use payroll adjustments when pay needs correction rather than trying to edit historical tap events.
