# Insurance Policies

Insurance policies control premiums, coverage, and claim rules.

## Where to work
Go to **Bills > Insurance**.

## Create or update a policy
Use **Add New Policy** to create coverage or edit an existing policy in **Existing Policies**.

## Key policy fields
- Premium and frequency.
- Claim type and waiting period.
- Maximum claims and status.

## Student enrollment
Use **Active Student Policies** to see who is enrolled and manage coverage.
