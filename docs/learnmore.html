Transition details

Classroom Token Hub V1 has been sunset. Between the V1 cutoff and the V2 launch, only this page and the transition landing page remain publicly reachable.

What changed

  • All V1 sign-in, dashboard, API, and admin paths are disabled.
  • All apex-domain traffic redirects to the transition page.
  • Subdomains no longer expose the Flask application.

What stays online

  • /v2transition.html
  • /learnmore.html

What to expect

  • V2 launch information will be published from the same transition surface.
  • Users should not expect classroom data entry or sign-in during this window.
