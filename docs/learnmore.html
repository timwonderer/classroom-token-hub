Why Classroom Token Hub was built

Classroom Token Hub was originally developed as a personal project to manage my classroom economy. While there are a few others existing platforms, I couldn't find one that has all the features I need along with the transparency I want my student to see. I also find most platforms design their user interface to be either too childish or too sterile.

Core Features

Classroom Token Hub comes with everything you need to run a robust classroom economy without the headache.

  • insights

    Analytics Dashboard

    System health metrics with participation tracking, giving you actionable insights to optimize your class economy.

  • home

    Classroom Rent

    Charge rent for desks or privileges. Setup per-use, time-based, or hall pass rent items with automatic coverage tracking.

  • payments

    Automated Payroll

    Configurable pay rates, rewards, and fines. Run payroll automatically or manually based on student attendance.

  • storefront

    Store Management

    Offer virtual perks or physical items. Set expiration dates, manage inventory, and easily track student redemptions.

  • badge

    Hall Pass Approvals

    Manage time-limited passes with automatic tracking and approvals, directly tied into the classroom economy.

  • account_balance

    Banking Simulation

    Manage checking and savings accounts with configurable interest rates, transfers, and complete transaction histories.

  • schedule

    Attendance Tracking

    Automatic time logging with a "Clock In / Break" system. Earn your base pay by simply participating in class.

  • storefront

    Classroom Store

    Use your hard-earned tokens to purchase virtual perks, rent privileges, or physical items from your teacher's store.

  • shield

    Insurance & Claims

    Learn real-world risk management by purchasing optional insurance policies to cover unexpected classroom fines.

  • phone_iphone

    Mobile Friendly App

    Installable directly on iOS and Android devices. Keep track of your finances on the go with a seamless mobile experience.

Our Architecture

Surprises are great for parties, but not for software that classrooms depend on. We've built the foundation to ensure that your classroom economy runs smoothly, reliably, and securely. Every archiectural decision can be traced back to a particular failure mode within our own classrooms that we have experienced first hand.

  • security

    Rock-Solid Reliability

    Whether students are purchasing items, transferring tokens, or claiming insurance, everything happens safely and accurately. You can trust the math.

  • group_work

    Strict Classroom Boundaries

    What happens in your class, stays in your class. We've strengthened privacy so that student data and activity is strictly limited to their enrolled periods.

  • speed

    Lightning Fast for Large Classes

    No more waiting for the roster to load. We've supercharged the app so everything loads almost instantly, even for classes with over 60 students.

  • lock_person

    Accident-Proof Design

    Ever deleted something by mistake? We added timed confirmation gates to prevent accidental data loss. We also automatically delete personal information when it's no longer needed.

  • architecture

    Safe and Predictable

    The system is designed with strict guardrails so things always work exactly the way you expect them to, with no surprising behaviors.

  • verified_user

    One Account, Multiple Classes

    Students only need a single login account to join multiple classes with different teachers. Each class economy runs independently, but logging in is simple.
