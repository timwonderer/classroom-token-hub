This is a preview route for testing the local Docusaurus handoff from Flask.

Current intent:

- prove that legacy Flask docs routes can hand off to Docusaurus selectively
- experiment with layout, navigation, and route stability
- avoid locking v2 documentation into a canonical structure too early

What this is not:

- a complete student manual
- the canonical v2 reference
- a commitment to the final route taxonomy beyond this pilot

While the v2 docs rework is still open, detailed and stable student guidance should continue to live in the repo docs tree and in-product help flows.
