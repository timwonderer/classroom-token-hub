How It Works
  • Pass Type Name: The destination students can select (e.g., Bathroom, Office, Nurse)
  • Queue Limit: Maximum number of students who can be waiting + currently out for this destination (leave blank for unlimited)
  • Simultaneous Limit: Maximum number of students who can be out at the same time for this destination (leave blank for unlimited)
