How It Works
  • Master Toggle: Enable or disable the entire hall pass system
  • Pass Type Toggle: Enable or disable individual pass types
  • Queue Limit: Maximum number of students who can be approved and waiting for this destination (leave blank for unlimited)
  • Simultaneous Limit: Maximum number of students who can be out at the same time for this destination (leave blank for unlimited)
  • Total Queue Limit: Overall cap for approved & waiting students across all pass types (only editable when any pass type has unlimited queue)
  • Total Simultaneous Limit: Overall cap for students currently out across all pass types (only editable when any pass type has unlimited out)
